### Futsal/Football table predictor
## Data sources
This page is designed to work with copy and pasted data from sources similar to the below:

**Competition Table** = 'View competition table' (e.g. https://www.futsalhq.com.au/pointscore/7636/16040)
- Copy and paste the entire table, including the headers (Team, Played, Wins, etc.)

**Upcoming Schedule** = Fixtures & Results page (e.g. https://www.futsalhq.com.au/games/7636/16040) -> 'Download'
- Copy and paste everything in the document, from the next upcoming round to the final round

## Updating the HTML
Nothing special about this, just make changes to the HTML code and commit the change
